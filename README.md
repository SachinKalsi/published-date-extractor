# published-date-extractor

A machine learning model which extracts the publication date from a given web URL, preferably a blog URL.

Tools used:
1. Beautiful Soup
2. scikit-learn
3. pandas
4. HTML Tag Annotator (https://github.com/SachinKalsi/html_tag_annotator)
